# i18next 자동화 테스트

### 개요

번역파일을 일일이 json파일로 치환하는 일련의 수동 작업들이 불편하여 구글링한 결과, 구글 시트를 이용하여 간편하게 자동화 방법을 찾게 되어 테스트 겸 진행

### build

```
-- 스캔
npm run scan

-- 업로드
npm run upload

-- 다운로드
npm run upload
```
